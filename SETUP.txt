ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА ORIENT METAL HR360

1. ПОДГОТОВКА
-------------
- Установите Node.js (версия 16 или выше)
- Установите Git
- Создайте пустую папку для проекта

2. КЛОНИРОВАНИЕ И УСТАНОВКА
--------------------------
1. Клонируйте репозиторий:
   git clone [URL репозитория]
   cd [название папки проекта]

2. Установите зависимости:
   npm install

3. НАСТРОЙКА GOOGLE SHEETS API
-----------------------------
1. Создайте проект в Google Cloud Console
2. Включите Google Sheets API
3. Создайте сервисный аккаунт и скачайте JSON-ключ
4. Поместите JSON-файл в папку server/ с именем:
   my-telegram-bot-project-431322-aa13ecf7fb96.json

4. НАСТРОЙКА ОКРУЖЕНИЯ
---------------------
1. Создайте файл .env в корневой папке:
   REACT_APP_API_URL=https://orienthr360.serveo.net
   PORT=3001

2. Создайте файл server/.env:
   PORT=3001
   GOOGLE_SPREADSHEET_ID=ваш_id_таблицы
   GOOGLE_SHEET_ID=ваш_id_листа

5. ЗАПУСК ПРОЕКТА
----------------
1. Запустите React-приложение:
   npm start

2. В новом терминале запустите сервер:
   cd server
   node server.js

3. В третьем терминале создайте туннель:
   ssh -R orienthr360.serveo.net:80:localhost:3001 serveo.net

6. ПРОВЕРКА РАБОТОСПОСОБНОСТИ
---------------------------
1. Откройте браузер и перейдите по адресу:
   http://localhost:3000

2. Проверьте доступность через туннель:
   https://orienthr360.serveo.net

7. ВАЖНЫЕ МОМЕНТЫ
----------------
- Убедитесь, что все три процесса (React, сервер, туннель) запущены одновременно
- Таблица Google Sheets должна быть доступна для записи сервисному аккаунту
- При первом запуске может потребоваться подтверждение доступа к Google Sheets

8. СТРУКТУРА ТАБЛИЦЫ
------------------
Таблица должна содержать следующие столбцы:
A: Компания
B: Отдел
C: Должность
D: Руководитель
E: Коммуникация (баллы)
F: Устойчивость (баллы)
G: Задачи (баллы)
H: Коммуникация (%)
I: Устойчивость (%)
J: Задачи (%)
K: Общие баллы
L: Общий процент
M: Дата

9. УСТРАНЕНИЕ ПРОБЛЕМ
-------------------
Если возникают проблемы:
1. Проверьте все логи в консоли
2. Убедитесь, что все процессы запущены
3. Проверьте доступность сервера через curl или Postman
4. Проверьте права доступа к Google Sheets

10. ДОПОЛНИТЕЛЬНО
---------------
- Для остановки всех процессов используйте Ctrl+C
- Для перезапуска сервера: node server.js
- Для пересборки React: npm run build
